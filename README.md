# Pastimes - ClothingStore – E-commerce Marketplace with Messaging & Appeals


> Group Members: Wasama Makolo & Cameron Reece Johannes

A complete PHP + MySQL web application for a clothing store. Buyers can browse items, chat with sellers, and checkout. Sellers can list clothing and manage requests. Admins have full CRUD control over users/items, can remove listings with reason, and review seller appeals.

## **Key Features**

1. **Customer Shopping:** Add to cart, update quantities, remove items, checkout, continue shopping.
2. **Admin Management:** Add, delete, update clothing items and user accounts.
3. **Seller Onboarding:** Sellers submit requests with clothing description, image, and brand.
4. **Admin Communication:** Admin can directly message sellers and buyers to coordinate delivery/quality.

------------------------

### **Extended Features**
5. **Private Messaging:** Buyers message sellers from product pages. Threads are locked to that item.
6. **Access Control:** Sellers can only reply to buyers who messaged them first, or to admin.
7. **Removal Notices:** Admin can remove sellers/items with a written reason. System auto-sends message + notification.
8. **Appeals System:** Removed sellers/items get a rebuttal link. They can submit appeals which admin reviews in dashboard.
9. **Notifications:** Real-time unread message count in navbar for all users.
10. **Responsive UI:** Clean navigation, styled forms, mobile-friendly product grid.
